 FREEWARE: H2D v1.14 - C headers to Modula-2 definition modules translator



Source:       xTech Ltd.

                H2D v1.14 is available

                Copyright (c) 1996 xTech Ltd.

OS/2 and Windows NT/95 Editions of H2D v1.14 are now available.
Linux edition is about to be released soon.

H2D is an utility which translates C header files (*.h) into Modula-2
definition modules. It enables a Modula-2 programmer to use the huge
set of free, shareware, and commercial libraries, which interfaces are
usually defined in terms of C language, as well as APIs of the most
popular operating systems and products.

We distribute H2D as freeware in order to encourage more programmers
to use Modula-2 and Oberon-2.

H2D is intended to be used with our XDS family of Modula-2/Oberon-2
systems, but it can be easiliy adapted to other Modula-2 compilers.
